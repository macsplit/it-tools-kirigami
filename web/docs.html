Tools reference

This page covers install commands, interface structure, and the full tool list currently defined in the source tree.

Purpose

Tools is a Linux desktop app for developer and IT support tasks: format conversion, encoding, hashing, parsing, text cleanup, and quick reference utilities.

Current scope

The current build contains 48 tools in 6 categories: Crypto, Converter, Network, Web, Development, and Text.

Interface

How it is laid out

1

Sidebar

Use the left sidebar to switch categories and search for a tool by name.

2

Dashboard

The dashboard shows the available tools as cards for quick scanning.

3

Tool page

Each tool opens in the main pane and provides its own inputs and output area.

4

Use

Most tools are direct transformations: paste input, adjust options if needed, then copy the result.

Install

Flatpak install options

Direct install

app.pastetools.Tools.flatpakref

Repository file

pastetools.flatpakrepo

Bundle download

app.pastetools.Tools.flatpak

flatpak install https://pastetools.app/app.pastetools.Tools.flatpakref
flatpak remote-add --if-not-exists pastetools https://pastetools.app/pastetools.flatpakrepo
flatpak install pastetools app.pastetools.Tools
