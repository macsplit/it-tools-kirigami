Getting started with Tools

This guide explains what the app does, who it is for, and where to find the utilities most people use first.

  • app.pastetools.Tools

  • License: GPL-3.0-or-later

Overview

Tools is a Linux desktop app that keeps many small technical utilities in one place.

Who it is for

It is useful for developers, sysadmins, support engineers, QA, and anyone who works with technical text and data.

Using the App

How to work with Tools

1

Open the app

Launch Tools from your application menu on Linux.

2

Browse by category

Pick from Crypto, Converter, Network, Web, Development, or Text.

3

Enter your input

Most tools take a value or text block and show the result straight away.

4

Copy the result

Take the result and move on.

Install

Flatpak install options

Direct install

app.pastetools.Tools.flatpakref

Repository file

pastetools.flatpakrepo

Bundle download

app.pastetools.Tools.flatpak

flatpak install https://pastetools.app/app.pastetools.Tools.flatpakref
flatpak remote-add --if-not-exists pastetools https://pastetools.app/pastetools.flatpakrepo
flatpak install pastetools app.pastetools.Tools

Tool Guide

Main categories and examples

Crypto

Generate random tokens, create hashes, and calculate HMAC values for quick security and debugging tasks.

Converters

Move between Base64, integer bases, colors, JSON, YAML, XML, CSV, Markdown, temperature units, and more.

Network

Inspect IPv4 data, calculate subnets, generate random ports, build WiFi QR codes, and create MAC addresses.

Web

Work with URL encoding, HTML entities, JWT parsing, URL inspection, HTTP status references, and basic auth strings.

Development

Format JSON, XML, and SQL; generate UUIDs; inspect MIME types; parse cron expressions; and calculate chmod values.

Text

Convert case styles, generate placeholder text, measure text statistics, build slugs, and spell text with the NATO alphabet.
