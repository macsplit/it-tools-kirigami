What you can do

  • Format JSON, XML, and SQL
  • Generate tokens, UUIDs, hashes, and HMAC values
  • Convert Base64, CSV, JSON, XML, and YAML
  • Work with URLs, JWTs, MIME types, and HTTP codes
  • Handle IPv4, subnets, ports, MAC addresses, and WiFi QR codes

Included Categories

A single app with a wide toolbox

Crypto

Token generation, hashes, and HMAC values for quick security-related tasks.

Converters

Common format and value conversions, including JSON, XML, CSV, Base64, Markdown, and more.

Network

IPv4 conversion, subnet calculations, random ports, MAC addresses, and WiFi QR generation.

Web

URL encoding, parsing, JWT inspection, HTML entities, basic auth strings, and status code lookup.

Development

UUIDs, MIME types, formatters, SQL prettifying, chmod help, and cron expression parsing.

Text

Case conversion, text statistics, slugification, NATO spelling, obfuscation, and placeholder text.

Install

Install the Flatpak

Direct install

Install from the hosted ref file.

app.pastetools.Tools.flatpakref

Repository file

Add the PasteTools Flatpak repository to Flatpak.

pastetools.flatpakrepo

Bundle

Download a Flatpak bundle directly.

app.pastetools.Tools.flatpak

flatpak install https://pastetools.app/app.pastetools.Tools.flatpakref
flatpak remote-add --if-not-exists pastetools https://pastetools.app/pastetools.flatpakrepo
flatpak install pastetools app.pastetools.Tools
