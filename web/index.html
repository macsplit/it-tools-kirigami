Linux Desktop Utility Suite

48 local utility tools in one Linux app.

Tools groups common conversion, formatting, networking, text, and reference utilities into a single Qt and Kirigami application for developers and IT admins.

What It Covers

Tool coverage

Crypto

Token generation, hashing, HMAC, and bcrypt password hashing.

Converters

Base64, integer bases, colors, JSON and XML transforms, CSV, Markdown, date and time, temperature, lists, binary, and Roman numerals.

Network and web

IPv4 tools, WiFi QR payloads, MAC addresses, ports, URL helpers, HTML entities, JWT parsing, and HTTP status reference data.

Development and text

MIME types, UUIDs and ULIDs, JSON/XML/SQL formatting, chmod, cron parsing, percentage math, case conversion, text statistics, slugify, numeronyms, Unicode, and NATO alphabet.

Install

Install the Flatpak

Direct install

Install from the hosted ref file.

app.pastetools.Tools.flatpakref

Repository file

Add the PasteTools Flatpak repository to Flatpak.

pastetools.flatpakrepo

Bundle

Download a Flatpak bundle directly.

app.pastetools.Tools.flatpak

flatpak install https://pastetools.app/app.pastetools.Tools.flatpakref
flatpak remote-add --if-not-exists pastetools https://pastetools.app/pastetools.flatpakrepo
flatpak install pastetools app.pastetools.Tools
