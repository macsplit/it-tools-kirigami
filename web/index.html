Tools
GitHub

54 utility tools in one Linux app.

A native Linux application grouping common conversion, formatting, networking, text, and reference utilities into a single interface. Fast, offline, and secure.

Install Flatpak View Source
Available Tools
Crypto
Token Generator, Hash Text, HMAC Generator, Bcrypt Password Hash.
Converter
Base64, Integer Base, Color, JSON/YAML/XML/CSV/TOML transforms, Text to Binary, Epoch, Date/Time, Roman Numerals, Temperature, List, Markdown.
Network
IPv4 Converter, Subnet Calculator, WiFi QR, MAC Generator, Random Port Generator, IPv4 Range Expander.
Web
Basic Auth, URL Encode, HTML Entities, JWT Parser, URL Parser, HTTP Status Codes.
Development
MIME Types, UUID/ULID, JSON/XML/SQL formatters, Chmod, Cron Parser, Percentage Calculator, Math Evaluator.
Text
Case Converter, Email Normalizer, NATO Alphabet, Numeronym, Unicode, Lorem Ipsum, Text Stats, Slugify, Obfuscator.

Installation
flatpak install https://pastetools.app/app.pastetools.Tools.flatpakref

Or add the repository:

flatpak remote-add --if-not-exists pastetools https://pastetools.app/pastetools.flatpakrepo
flatpak install pastetools app.pastetools.Tools
