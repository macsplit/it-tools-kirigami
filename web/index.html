40 small tools in one Linux app.

Tools groups common conversion, formatting, networking, text, and reference utilities into a single Qt and Kirigami application.

What It Covers

Tool coverage

Crypto

Token Generator, Hash Text, HMAC Generator.

Converters

Base64, integer bases, colors, JSON, YAML, XML, CSV, Markdown, temperature, lists, binary, Roman numerals.

Install

Install the Flatpak

Direct install

Install from the hosted ref file.

app.pastetools.Tools.flatpakref

Repository file

Add the PasteTools Flatpak repository to Flatpak.

pastetools.flatpakrepo

Bundle

Download a Flatpak bundle directly.

app.pastetools.Tools.flatpak

flatpak install https://pastetools.app/app.pastetools.Tools.flatpakref
flatpak remote-add --if-not-exists pastetools https://pastetools.app/pastetools.flatpakrepo
flatpak install pastetools app.pastetools.Tools
